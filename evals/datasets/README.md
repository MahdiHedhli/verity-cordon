# Verity Cordon Synthetic Evaluation Dataset

`memory-poisoning-fixtures-v1.json` is an original, synthetic dataset for
exercising Verity Cordon's offline fixture pipeline. It contains no copied
benchmark material, real credentials, personal data, or production content.

The samples cover benign facts and preferences, obvious and indirect attacks,
persistence attempts, poisoned tool output, cross-task contamination,
self-reinforcement, false-positive traps, and synthetic secret handling.

The dataset is licensed under Apache-2.0. See the repository's top-level
`LICENSE` file. Source and license metadata are also embedded in the dataset.

Results produced from this dataset describe only these curated fixtures. They
must not be presented as universal detection accuracy, production efficacy, or
live-model performance.
