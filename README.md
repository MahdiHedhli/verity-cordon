# Verity Cordon

**A tamper-evident memory firewall for Codex.**

**Verifiable memory. Revocable trust.**

Verity Cordon protects Codex from persistent memory poisoning by making durable
memory explicit, attributable, policy-governed, and revocable. This repository
is under active OpenAI Build Week development; the judge quickstart will be
published here only after the corresponding path has been verified.

The feature source of truth is
[`specs/001-codex-memory-firewall/`](specs/001-codex-memory-firewall/). Security
claims and limitations are defined in
[`docs/security/`](docs/security/), and donor prior-art analysis is recorded in
[`docs/hackathon/BASELINE_COMPARISON.md`](docs/hackathon/BASELINE_COMPARISON.md).

Verity Cordon does not claim factual truth verification, tamper-proof storage,
complete prompt-injection prevention, undocumented Codex interception, or
protection from a compromised host.
